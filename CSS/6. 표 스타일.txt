1. caption-side
    caption-side : top | bottom
    - 표 제목의 위치 지정

2. border
    table{
        border: 1px solid black
        }
    td, th{
        border: 1px dotted border
    }
    - 표의 테두리 지정 table{} 표 전체 테두리
                       td,th{} 셀 테두리

3. border-spacing
    border-spacing : 수평거리 수직거리
    - 셀 사이 여백 지정

4. border-collapse
    border-collapse : collapse | seperate
    - 표와 셀의 테두리를 합치거나 따로 표시함 
    - 디폴트는 seperate
