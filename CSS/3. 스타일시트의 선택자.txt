1. 전체 선택자
    - * {속성 : 값; ....}
        - 스타일을 문서의 모든 요소에 적용할 때 사용

2. 타입 선택자
	- 태그명 {스타일 규칙}
		- 해당 태그 요소에 적용할 때 사용

3. 클래스 선택자
	- .클래스명 {스타일규칙}
		- 문서의 일부에 적용하고 싶을 떄 사용
		- 적용하고자 하는 태그 안에 class="클래스명" 속성을 지정하여 사용
			- ex) </style>  
					.blue { color : blue}
					.accent {border : 1px solid #000}
				  </style>

				   <label class="blue accent"> 이름 : </label>
				   * 2개 이상 적용할 때는 공백으로 구분

4. ID 선택자
	- #아이디명 {스타일 규칙}
		- 문서에서 딱 한 번만 적용 가능(중복사용 불가)
		- 주로 문서의 레이아웃과 관련된 스타일을 지정하거나 
		  웹 요소에 자바스크립트 프로그램을 사용하면서 요소를 구별할 때 사용
		- <div> 태그와 함께 사용하면 레이아웃 지정에 유용

5. 그룹 선택자
	- 선택자1, 선택자2 {스타일 규칙}
		- 2개 이상의 선택자의 스타일 규칙을 한번에 정의할 때 사용
			- ex) h1, p {text-align : center;}
				 	=> h1, p 요소를 중앙 정렬	

6. 스타일의 우선순위
	- 사용자 스타일 > 제작자 스타일 > 브라우저 기본 스타일

	- 제작자 지정 스타일의 우선순위
		1) !important (해당 속성의 끝에 추가)
		2) 인라인 스타일
		3) id 스타일
		4) 클래스 스타일
		5) 타입 스타일
