1. 연결 선택자
    1) 하위 선택자
        상위요소 하위요소 { }
        - 상위요소에 있는 모두 하위 요소들의 스타일을 지정
        - 하위 요소의 더 하위요소에까지 모두 적용
        ex) section p { }
            - section에 있는 p요소들의 스타일 지정  
    2) 자식 선택자
        부모요소 > 자식요소 { }
        - 자식요소의 스타일을 지정
        - 자식요소의 하위요소에는 적용되지 않음
    3) 인접 형제 선택자
        요소1 + 요소2 { }
        - 부모요소가 같은 자식요소끼리의 관계
        - 요소 1 이후에 가장 먼저 오는 요소2의 스타일 지정
    4) 형제 선택자
        요소1 ~ 요소2 { }
        - 부모요소가 같은 자식요소끼리의 관계
        - 요소1 의 이후에 오는 모든 요소2의 스타일 지정
    5) [속성] 선택자
        요소1[속성] { }
        - 요소1중에서 [속성]이 있는 요소의 스타일 지정
    6) [속성=속성값] 선택자
        요소1[속성=속성값] { }
        - 요소 1 중에서 주어진 속성과 속성값이 일치하는 요소의 스타일 지정
    7) [속성~=값] 선택자
        [속성~=값] { }
        - 여러 속성값 중에서 해당 속성값이 포함된 요소의 스타일 지정
    8) [속성|=값] 선택자
        [속성|=값] { }
        - 특정 속성값이 포함된 속성요소의 스타일 지정
    9) [속성^=값]
        [속성^=값] { }
        - 특정 속성값으로 시작하는 속성요소의 스타일 지정
    10) [속성 $= 값]
        [속성$=값] { }
        - 특정한 값으로 끝나는 속성의 요소의 스타일 지정
    11) [속성*=값] 
        [속성*=값] { }
        - 일부 속성값이 일치하는 요소의 스타일 지정
